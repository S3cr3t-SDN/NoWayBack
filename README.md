# There Is NoWayBack

# Install:
```console
go install github.com/S3cr3t-SDN/NoWayBack@v1.0.0
```

# Usage:
```console
root@kali:~# NoWayBack domain.com
OR
root@kali:~# NoWayBack hosts.txt
```

You can run without installing
```console
git clone https://github.com/S3cr3t-SDN/NoWayBack.git
cd NoWayBack
go run NoWayBack.go
```
